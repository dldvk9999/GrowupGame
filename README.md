# 🐣 키우기 게임 (GrowupGame)

> 알에서 깬 지 5분 된 몬스터가, 5차 전직을 하고 나면 온 대륙이 벌벌 떠는 재앙이 됩니다.

방치하면 알아서 크고, 신경 쓰면 더 잘 크는 몬스터 육성 RPG예요. 불/물/풀 중 하나를 골라 계약을 맺고, **100개 챕터 × 1000개 스테이지**를 정복하면서 강해지는 게임입니다. 탭 하나로 즐기는 웹 게임이라 설치도 필요 없어요 (원한다면 PWA로 설치도 가능하지만요).

## 🔥 이런 게임이에요

- **키운다** — 스타터 몬스터 하나를 골라서 레벨업시키고, 3단계까지 진화시켜요. 방치만 해도 자동사냥으로 조금씩 크지만, 진짜 강해지려면 직접 스테이지에 도전해야 해요.
- **전직한다, 다섯 번이나** — 레벨 30/60/100/140/180마다 전직 던전이 열려요. 그냥 레벨업만으론 안 되고, 던전을 실제로 깨야 진짜 힘이 생겨요. 5차까지 다 찍으면 원래 캐릭터가 맞나 싶을 정도로 스탯이 달라져요(무려 16배).
- **뽑는다** — 상점에서 무기/방어구/장갑/신발, 그리고 스킬까지 전부 가챠로 뽑아요. 등급은 노멀부터 신화까지 5단계, 스킬은 데미지/힐뿐 아니라 기절·도트·버프·헤이스트까지 50종이나 있어요. 뽑은 스킬 편성은 별도 탭에서 여유롭게(슬롯은 스크롤해도 안 사라지게 고정돼있어요).
- **합성한다** — 강화수치 10을 모으면 한 단계 위 등급으로 합성할 수 있어요. 일괄합성 버튼 누르면 자동으로 갈 수 있는 데까지 쭉 가줘요.
- **싸운다, 사람이랑도** — PvP 경기장에서 다른 플레이어(또는 비슷한 세기의 가상 상대)와 붙어요. 투사체가 날아다니고 화면이 흔들리는 제법 그럴듯한 전투 연출도 있어요. 이기면 PvP 전용 재화를 벌고, 그걸로 코스튬도 살 수 있어요.
- **다 같이 용을 팬다** — 매주 전체 유저가 체력 하나를 공유하는 월드보스가 등장해요(지금 만나면 화면을 꽉 채울 정도로 거대해요). 하루 3번씩 부지런히 깎아서 일주일 안에 처치하면, 참여한 사람 전원한테 7일짜리 "용의 버프"(공격력·방어력 20배!)가 붙고 닉네임이 무지개색으로 반짝여요. 못 잡으면? 그래도 깎은 만큼 골드는 챙겨드려요.
- **떠든다** — 로비 채팅으로 다른 플레이어들이랑 실시간으로 수다도 떨 수 있어요(내가 접속한 동안의 대화만 보여요).
- **길을 잃지 않는다** — 화면 한켠에 항상 떠있는 가이드 미션이 "지금 뭘 해야 하는지" 계속 알려줘요. 전직할 때 되면 미션도 눈치채고 던전으로 등 떠밀어줘요.

## 📱 모바일도 신경 썼어요

작은 화면에서는 헤더가 햄버거 메뉴로 접혀요. 눌러보시면 오른쪽에서 스르륵 나타나요.

## ⌨️ 키보드파를 위한 배려

마우스 딸깍딸깍이 귀찮다면 숫자키 `1`~`9`로 스킬 쏘고, `Space`로 다음 스테이지 넘어가고, `Tab`으로 탭 이동, `G`/`Shift+G`/`Ctrl+G`로 뽑기 1/10/100연차까지 돌릴 수 있어요. 전체 단축키 목록은 [`harness/ui-and-ux.md`](./harness/ui-and-ux.md)에 있어요.

## 🛠️ 기술 스택

Vite + React 18 프론트엔드, Supabase(Postgres + Auth + Realtime) 백엔드. 골드 지급/뽑기 확률/전직 조건 같은 핵심 로직은 전부 서버(SQL RPC)에서 최종 검증하려고 노력하고 있어요 — 실제로 이 프로젝트는 몇 차례 보안 재점검을 거치면서 "devtools로 RPC를 직접 두드리면 뚫리는" 구멍들을 찾을 때마다 바로 막아왔어요. 다만 전투 판정 자체는 아직 클라이언트에서 계산되는 구조라 완벽하진 않고, 이런 트레이드오프와 발견/수정 이력은 [`harness/security.md`](./harness/security.md)에 숨김없이 다 적어뒀어요.

## 📚 더 자세히 알고 싶다면

이 게임의 모든 시스템, 밸런스 공식, DB 스키마, 버그/보안 수정 이력까지 전부 [`harness/`](./harness) 폴더에 카테고리별로 정리되어 있어요. 개발 이어서 하고 싶으면 [`harness/README.md`](./harness/README.md)부터 시작하면 돼요.

## 🚀 로컬에서 돌려보기

```bash
npm install
cp .env.example .env   # VITE_SUPABASE_URL, VITE_SUPABASE_ANON_KEY 채우기
npm run dev
```

Supabase 마이그레이션 적용 방법은 [`harness/dev-guide.md`](./harness/dev-guide.md) 참고하세요.
